This project is my attempt to use automation and AutoHotKey script(s) to learn and document the value of each item modifier in Path of Exile 2. The idea is to search up each item mod individually on the Trade Market to determine which item mods are desirable and expensive. To determine which item mod(s) currently exist in the game for a given type of item, we will parse raw data from the website poe2 database website. Due to there being hundreds of mods in the game, we are using automation to look up and store each individual price data point from the Trade Market (which is where the AHK script comes into play).

This is a WIP and this README will be updated as the project progresses further along.
